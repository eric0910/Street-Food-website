# Street-Food-website
# Description.
This is a Website that  contains information about food and also have a section on how you can reach us.
The landing page gives access to navigate all the offered services and provided snacks.
 # Author.
 Eric Kithinji.

# technology.
The website is built using HTML and styled using CSS.
HTML 5 is the version used.
# pre-requisites.
To use the website you should have at least a free space in your computer.
Ensure your browser is effective in displaying the website well.
# Installation.
Sign up to your GitHub account.
If you don't have a GitHub account create one.
Search eric0910.
Click on the Street-Food website.
Deploy and the website will open in your browser.
